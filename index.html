Project proposal

the object of this project is to demonstarate our devlopment skills and in both frontend and backend, therefore the idea is not that creative. A polls website is a website that a user can vote for polls made by other users. and make his own polls and let the users to vote on his poll, he also can share his poll link so unregistered users can vote using his link.

Features

  1. User login System
  2. Users can create new polls .
  3. The system shall provide real time result screen for the users.
  4. Users can set deadlines for their poll.
  5. The System shall provide unique URL for each poll
  6. Any user with a valid URL can vote
